# **App Name**: LunaWell

## Core Features:

- Dashboard Overview: Display a personalized dashboard summarizing key health metrics and wellness activities for the day.
- Health Tracking Input: UI to track various health metrics (sleep, mood, symptoms).
- Water Intake Logger: Allow users to log their daily water intake with visual progress.
- Skincare Routine Planner: Enable users to create and track their skincare routine (morning and evening).
- Daily Reminder Scheduler: Allow users to schedule and manage daily reminders for medications, appointments, or self-care activities.
- Progress Reports Generator: Generate reports visualizing progress in health tracking, water intake, and skincare routines.
- AI-Powered Health Insights Tool: AI-powered chatbot that answers women's health, wellness & self-care questions by synthesizing credible, high-quality medical information.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a calming and elegant feel.
- Background color: Light beige (#F5F5DC) to create a warm and inviting atmosphere.
- Accent color: Pale rose (#FFB9C6) for highlighting important information and interactive elements.
- Body and headline font: 'Alegreya', a serif font, for a refined, contemporary, intellectual feel
- Use delicate and feminine icons to represent different health aspects and activities.
- Maintain a clean and spacious layout with gentle curves to enhance user experience.
- Incorporate subtle transitions and animations for feedback and engagement.
